# Dither Studio

A web-based realtime dithering application. Point your camera at the world and watch it transform through classic dithering algorithms, retro palettes, and experimental particle physics.

## Features

- **Live camera dithering** — realtime processing at 30fps
- **15+ dithering algorithms** — Floyd-Steinberg, Atkinson, Bayer, blue noise, halftone, and more
- **20+ retro palettes** — Game Boy, NES, CGA, PICO-8, C64, and custom palettes
- **Mass-preserving particle transport** — experimental physics mode where pixels are particles that flow to their targets
- **Snap photos & record loops** — capture stills or short video loops
- **Export everywhere** — PNG, MP4, optimized presets for Twitter/Instagram/Discord

## Quick Start

Open `index.html` in a modern browser. No build step, no dependencies.
